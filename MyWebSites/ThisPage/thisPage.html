Web Projects Showcase

This page introduces the Web Projects Showcase and explains its purpose. It is designed to provide transparency about my web development work, demonstrating how I integrate multiple technologies across frontend and backend development. The showcase highlights responsive design, dynamic content, and database-driven features that together create an interactive, user-friendly experience.

Design and Structure

Dynamic Content

The page uses a lightweight custom database to store project information. On each load, project links and details are fetched dynamically. This database-driven approach—using SQLite—makes updates easy: changing the database entries automatically updates the page without rewriting HTML.

Hexagonal Layout

Projects are presented in a multi-layered hexagonal tile design. Each tile contains a project link and brief description. This hexagon-style interface adds a modern, organized look and helps distinguish each project visually. The tile design also makes the layout responsive: tiles rearrange gracefully on different screen sizes.

Responsive Design

The site uses responsive web design techniques to ensure usability on any device. CSS media queries adjust the layout and typography for desktop, tablet, or smartphone screens, giving a consistent user experience on all devices. For example, the burger (“hamburger”) menu appears on smaller screens to save space, while the footer and content tiles scale fluidly.

Navigation

A fixed header provides a burger navigation menu. From this menu, users can easily navigate to Home, Portfolio, and other sections of the site (such as Cybersecurity, Linux Scripting, and Metadata). The footer likewise contains quick links to About Me, Home, Portfolio, This Page, and Contact. These links make it easy to jump between main sections of the portfolio from anywhere on the site.

Security

User interactions are designed with security and clarity in mind. Login and subscription data are handled through secure processes: passwords are stored using industry-standard one-way hashing (rather than reversible encryption). Interactive elements like logout confirmations and password-change forms use clean modal dialogs to guide the user. All these design choices create a simple, clean interface that still follows good security practices.

Featured Projects

Raspberry Pi – CCTV

A single-camera solution with Raspberry Pi backend and Java/SQL/JavaScript frontend. Users can rename the camera in real time, toggle fullscreen, and log out via a modal. A lightweight database stores subscriptions and settings; passwords use one-way hashing. This prototype currently streams a sample webcam feed, but it is designed so that a real Raspberry Pi camera can be integrated seamlessly in the future.

Vision Scope (CCTV Dashboard)

An interactive CCTV management dashboard developed with Java, SQL, JavaScript (AJAX), HTML, and CSS. Users can dynamically add or remove camera feeds by entering a camera URL and a custom name (e.g. “Front Door Camera”). The interface is intuitive: camera names can be edited, unwanted cameras deleted individually or in bulk via friendly modal dialogs. The dashboard also shows personalized user information: the user’s avatar (uploadable by the user), their last login timestamp, and the total number of registered users.

Music Galaxy

A music dashboard with genre sorting, pagination, and light/dark themes. Built with PHP, JavaScript, and SQL. Plans include Spotify API integration. All user data is encrypted for privacy.

The Fly Now

An air-travel booking platform using PHP and JavaScript. Users can create accounts, search flights, book tickets, and manage bookings. The design is fully responsive, ensuring usability on all devices.

HTML & PHP Examples – Tools Demonstration

A collection of code examples in HTML, PHP, JavaScript, CSS, and MySQL. Includes interactive widgets and database-driven pages, all designed responsively to showcase practical web development patterns.
